Bucles

Cuando queremos que un trozo de código se ejecute varias veces seguidas, utilizamos bucles.

En JavaScript tenemos los siguientes:

while(condición) { acción }
Repite un bloque de código siempre que la condición previa se cumpla.
do { acción }
while(condición)
Igual que el anterior, pero primero recorre el código y después comprueba la condición para volver a recorrerlo.
for(condición) { acción }
Ejecuta un bloque de código un número de veces. Normalmente, se utiliza un iterador en la condición.
for(clave in objeto) { acción }
Como el anterior, pero el iterador será una propiedad de un objeto dado.
for(variable of iterable) { acción }
Otro subtipo de for en el que el iterador es un valor del objeto iterable dado.

Todos los tipos de bucles se pueden utilizar indistintamente, aunque será preferible uno u otro según el caso. Por ejemplo, para recorrer un vector, es más intuitivo y eficiente utilizar un for que un while.



Saltos

  • Cuando queremos salir de un bucle, utilizamos la instrucción break.
  • Cuando queremos saltar una iteración dentro de un bucle, utilizamos la instrucción continue.
