Condicionales

Muy a menudo, cuando escribe código, desea realizar diferentes acciones para diferentes decisiones. Para ello sirven las declaraciones condicionales.

En JavaScript tenemos las siguientes:

  • Se usa if para especificar un bloque de código que se ejecutará si una condición se cumple.
  • Se usa else para especificar un bloque de código a ejecutar si la condición declarada en el if anterior no se cumple.
  • Se usa else if para especificar una nueva condición, que se evaluará cuando la condición del if anterior sea falsa.
  • Se usa switch para especificar varios bloques alternativos de código para ejecutar.
